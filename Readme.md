# Python Backup System

## usage:

```bash
python backup.py -src="Path_to_Source_Folder" -dst="Path_To_Destination_Folder" -name="Archive_name.zip"
```

## Example for Windows
```shell script
python .\backup.py -src="C:\\USERS\\Oleg\\Downloads" -dst="c:\\Users\\oleg\\Desktop\BackUP" -name="backup.zip"
```
